Description

Detecting Glaucoma using Self Organising Maps

A self-organizing map (SOM) is a type of neural network which is trained using unsupervised learning to produce a low-dimensional, discretized representation of the input space of the training samples.

Dimensionality reduction property of SOM is used to segment retina cup

This was my project under Dr.K.M.M Rao while at BITS-Pilani

Details

Processing done in seven stages
  1. Green channel
  2. Gaussian matched filter
  3. OTSU filter
  4. Morphological closing
  5. Sobel filter & Thresholding
  6. Hough circles
  7. Self-organising maps
Output is Optic cup-to-disk ratio which can be used to determine glaucoma.
